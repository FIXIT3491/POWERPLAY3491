---------
GamePad 1
---------
Left Joystick X = Turning

Left Joystick Y Forwards/Backwards

Right Joystick X = Strafing

A = Change Driving Mode to Field Oriented

B = Change Driving Mode to Default

---------
GamePad 2
---------
Left Bumper/Right Bumper = Grabber

X = Floor

Y = Low Pole

B = Mid Pole

A = Hi Pole
